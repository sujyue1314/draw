# VoiceCanvas AI - 演示脚本

## 前置准备

1. 复制 `.env.example` 为 `.env.local`，填入 DashScope API Key
2. `pnpm install` 安装依赖
3. `pnpm dev` 启动开发服务器
4. 打开 Chrome/Edge 浏览器（需要 Web Speech API 支持）
5. 允许麦克风权限

## 基础演示

### 1. 生成图片

点击麦克风按钮，说出：
> "生成一只小猫坐在月亮上"

预期：
- 状态变为"思考中"
- 画布出现生成的图片
- 右侧面板显示组件列表（猫、月亮）
- TTS 播报"已生成画面，包含：猫、月亮。"

### 2. 增量添加

> "加一只小鸟在天上"

预期：
- 图片更新，包含小鸟
- 组件列表新增"小鸟"

### 3. 撤销

> "撤销上一步"

预期：
- 图片恢复到添加小鸟之前
- 组件列表移除"小鸟"
- 历史面板显示操作记录

### 4. 重做

> "恢复刚才的操作"

预期：
- 图片恢复包含小鸟的状态

### 5. 比例切换

> "改成 1:1"

预期：
- TopBar 比例标签变为"1:1"
- 后续生成的图片使用 1:1 比例

### 6. 复合操作

> "删掉月亮再加一艘船"

预期：
- 月亮被移除
- 船被添加
- TTS 播报两条操作结果

### 7. 多画布

> "新建画布"

预期：
- 底部出现新 tab
- 自动切换到新画布

> "切换到画布 1"

预期：
- 切换回第一个画布

## 手动操作演示

### 撤销/重做按钮

1. 生成图片 → 添加组件
2. 点击 TopBar 撤销按钮（←）
3. 点击 TopBar 重做按钮（→）

### 比例下拉

1. 点击 TopBar 右侧比例标签
2. 选择不同比例
3. 验证后续生成图片使用新比例

### 画布 Tab

1. 点击"+"新建画布
2. 点击不同 tab 切换画布

## 边界情况

### 空画布操作

> "撤销"

预期：TTS 播报"没有可以撤销的操作。"

### 不存在的组件

> "删除不存在的东西"

预期：TTS 播报"没有找到「不存在的东西」这个组件。"

### 无图片时编辑

> "把背景改成蓝色"（画布无图片）

预期：TTS 播报"当前画布没有图片，请先生成一张。"
